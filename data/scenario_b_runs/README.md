# Scenario B Runs

Scenario B is the limited-simulation optimization arm for the paper comparison.
These runs are generated separately from the original 177-case training corpus.

Store local result payloads and manifests here. Payloads are ignored by git.

Recommended manifest fields:

- Fixed geology/case-study version
- Operational parameters sampled
- MRST run ids and source paths
- Objective values and constraints
- Date generated
- Notes on failed or rejected runs
