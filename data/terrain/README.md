# Terrain Data

Place SRTM or equivalent terrain rasters here. Raster payloads such as `.tif`
and `.tiff` are intentionally ignored by git.

Expected use: route elevation sampling for the pipeline/logistics module.

Provenance to record when added:

- Source/download URL or local source path
- Coverage/bounds
- CRS and vertical datum
- Resolution
- Acquisition/download date
